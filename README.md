# K. Frantz GmbH Website

A Next.js website for K. Frantz GmbH, a German plumbing and heating company.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app router pages
- `components/` - React components (Header, Navigation, Footer, Banner)
- `public/images/` - Static images
- `styles/` - Global CSS styles

## Pages

- `/` - Start page with banner and contact information
- `/heizungsbau` - Heating construction services page
- `/heizungsrechner` - Heating calculator (placeholder)
- `/baeder` - Bathrooms (placeholder)
- `/gas-wasser-installation` - Gas-water installation (placeholder)
- `/jobs` - Jobs (placeholder)
- `/ausbildung` - Training/Apprenticeship (placeholder)
- `/kontakt` - Contact page

## Build

To create a production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```
